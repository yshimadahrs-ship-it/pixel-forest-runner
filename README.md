# Pixel Forest Runner

添付画像の雰囲気をもとに作った、ブラウザで遊べるレトロ風2Dジャンプアクションです。

- 紫髪のちびキャラを操作して、森と山のステージを進みます
- コインと星を集め、キノコ型モンスターを踏んで倒します
- 旗までたどり着くとクリアです
- PC / スマホ両対応

## 操作

- 移動: ← → / A D
- ジャンプ: ↑ / W / Space
- スマホ: 画面下のボタン

## ローカル起動

```bash
python3 -m http.server 4173
```

Then open <http://127.0.0.1:4173/>.
